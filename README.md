# urirun-connector-vdisplay

Native `vdisplay://` URI surface over the [vdisplay](../../wronai/vdisplay) package: windows/monitors/session enumeration (multi-backend, each with an `nl` description) and display diagnosis, so a urirun node / flow / readiness kernel can consume the environment state instead of re-deriving it ad hoc.

See `../urirun/docs/RESOLVE_FIRST_CONTROL_PLANE.md`.
